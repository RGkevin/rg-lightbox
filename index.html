    RG Lightbox

    A simple and lightweight javascript module that allows you to overlay images on the top of your webs. [I'm working on mobile friendly feature]

    No jQuery needed!

    Installation

    Using Bower

    $ bower install rg-lightbox --save

    Downloading source

    You can download the source code and then add the required files to your HTML

    Usage

    You have to include the two required files /dist/rg-lightbox.min.js and /dist/spin.js. Also you need to add the CSS file dist/rg-lightbox.css.

    Finally add to your main javascript file the following code

    var myCustomLightbox = new RGlightbox({/*options*/});

    RequireJS

    define(['rg-lightbox'], function(RGlightbox) {
  var myCustomLightbox = new RGlightbox({/*options*/});
});

    The HTML

    You just need to add the data-lightbox to bind the lightbox to your links.

    Creating galleries: just add the same value to the attribute.

    //creating individual images
  <a data-lightbox href="http://lorempixel.com/460/280/cats/" >
    <img src="http://lorempixel.com/200/100/cats/" alt="">
  </a>
  //creating galleries
  <a data-lightbox="my-gallery" href="http://lorempixel.com/460/280/cats/">
    <img src="http://lorempixel.com/200/100/cats/" alt="">
  </a>
  <a data-lightbox="my-gallery" href="http://lorempixel.com/460/280/cats/">
    <img src="http://lorempixel.com/200/100/cats/" alt="">
  </a>
  //another gallery
  <a data-lightbox="my-second-gallery" href="http://lorempixel.com/460/280/cats/">
    <img src="http://lorempixel.com/200/100/cats/" alt="">
  </a>
  <a data-lightbox="my-second-gallery" href="http://lorempixel.com/460/280/cats/">
    <img src="http://lorempixel.com/200/100/cats/" alt="">
  </a>

    Options

    Transition [number]

    Time in milliseconds of the CSS property -transition

    transition: 350 //default

    prevText [String]

    Inner text for 'Prev button'

    prevText: 'left' //default

    nextText [String]

    Inner text for 'Next button'

    nextText: 'next' //default

    Spin [object]

    RGlightbox uses the SpinJS library, so you can customize it using its own properties. *see spinJS documentation

    {
  lines: 13, // The number of lines to draw
  length: 20, // The length of each line
  width: 10, // The line thickness
  radius: 30, // The radius of the inner circle
  corners: 1, // Corner roundness (0..1)
  rotate: 0, // The rotation offset
  direction: 1, // 1: clockwise, -1: counterclockwise
  color: '#000', // #rgb or #rrggbb or array of colors
  speed: 1, // Rounds per second
  trail: 60, // Afterglow percentage
  shadow: false, // Whether to render a shadow
  hwaccel: false, // Whether to use hardware acceleration
  className: 'spinner', // The CSS class to assign to the spinner
  zIndex: 2e9, // The z-index (defaults to 2000000000)
  top: 'auto', // Top position relative to parent in px
  left: 'auto' // Left position relative to parent in px
}
