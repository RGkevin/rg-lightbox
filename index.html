    RG Lightbox

    A simple and lightweight javascript module that allows you to overlay images on the top of your webs. [I'm working on mobile friendly feature]

    Installation

    Using Bower

    $ bower install rg-lightbox --save

    Downloading source

    You can download the source code and then add the required files to your HTML

    Usage

    You have to include the two required files /dist/rg-lightbox.min.js and /dist/spin.js. Also you need to add the CSS file dist/rg-lightbox.css.

    Finally add to your main javascript file the following code

    var myCustomLightbox = new RGlightbox({/*options*/});
